# Retreat Booking API

This Flask-based API manages retreat bookings, offering endpoints for retreat information, bookings, and retreat creation.

## Features

- List and search retreats with pagination
- Book retreats with date conflict checking
- Create new retreats
- PostgreSQL database integration using SQLAlchemy

## Documentaion
- Postaman


## Prerequisites

- Python 3.7+
- PostgreSQL database

## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/Eshan276/printmaterial_mern.git
   cd retreat
   ```

2. Install the dependencies:

   ```sh
   pip install -r requirements.txt
   ```

## Running the Server

Start the server using:

```sh
python main.py
```
